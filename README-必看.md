## 🎯 这个软件是什么？

一个**隐蔽的小说阅读器**，适合工作时偷偷看小说。

### 核心功能

| 功能 | 说明 | 好处 |
|------|------|------|
| 🪟 透明窗口 | 可调节透明度（30%-100%） | 不容易被发现 |
| 👁️ 系统托盘 | 点击关闭隐藏到托盘 | 完全隐蔽 |
| 📌 窗口置顶 | 一键置顶 | 边工作边看 |
| 🎨 无边框 | 界面简洁 | 美观大方 |
| ⌨️ 快捷键 | Alt+H 快速隐藏 | 紧急时刻救急 |
| 📖 章节导航 | 上下章节切换 | 阅读流畅 |

## 🎬 软件使用演示

### 启动软件

```
1. 双击桌面上的"隐蔽小说阅读器"图标
2. 软件启动，显示欢迎界面
3. 拖动透明度滑块，调节到合适的透明度
```

### 正常阅读

```
1. 点击"加载示例"或直接看默认内容
2. 调节透明度到 70-85%
3. 点击"置顶"按钮
4. 把窗口放在工作文档旁边
5. 边"工作"边看小说 😊
```

### 紧急隐藏（老板来了！）

```
方法 1：按 Alt + H （最快）
方法 2：点击右上角 [◻] 按钮
方法 3：点击右下角 [🙈 快速隐藏] 按钮

✅ 窗口瞬间消失，只在托盘有个小图标
```

### 继续阅读（老板走了）

```
1. 看向 Windows 右下角
2. 点击小箭头（^）展开隐藏图标
3. 找到 📖 图标
4. 单击图标，窗口重新显示
```

### 退出软件

```
⚠️ 注意：点击窗口的 × 只是隐藏，不是退出！

正确方法：
1. 在托盘找到 📖 图标
2. 右键点击
3. 选择"退出"
```

## ✏️ 如何修改小说内容？

### 简单方法：直接编辑

1. 找到软件安装文件夹
   - 默认：`C:\Users\你的用户名\AppData\Local\Programs\secret-reader`
2. 用记事本打开 `index.html`
3. 找到这段代码（约第 200 行）：

```javascript
const chapters = [
    {
        title: "第一章 章节标题",
        content: `第一段内容。

第二段内容。`
    },
    // 添加更多章节...
];
```

4. 修改 `title` 和 `content`
5. 保存，重启软件

### 高级方法：重新打包

如果你想永久修改，重新打包：

1. 修改本地项目中的 `index.html`
2. 上传到 GitHub
3. 自动打包新版本
4. 下载安装新版本

## ❓ 快速问答

### Q: 这个软件安全吗？

**A:** 完全安全！代码开源，你自己打包，不涉及任何服务器上传或隐私泄露。

### Q: 会占用多少电脑资源？

**A:** 非常少！Electron 应用本身很轻量，大概占用 100-200MB 内存。

### Q: 可以在其他电脑上使用吗？

**A:** 可以！打包好的 exe 文件可以复制到任何 Windows 电脑上安装使用。

### Q: 打包失败怎么办？

**A:** 查看《打包教程-超详细版.md》的 FAQ 部分，里面有常见问题和解决方法。

### Q: 可以打包 Mac 和 Linux 版本吗？

**A:** 可以！修改 `.github/workflows/build.yml` 配置即可，教程中有说明。

### Q: 需要安装 Python、Java 等环境吗？

**A:** 不需要！只需要 Node.js（打包时 GitHub 云端会自动处理）。

### Q: 软件会被杀毒软件拦截吗？

**A:** 可能会，因为未签名的 exe 可能会被标记。选择"允许运行"即可。
