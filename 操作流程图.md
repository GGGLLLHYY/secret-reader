# 📊 操作流程图解

## 🗺️ 完整操作流程

```
开始
  ↓
┌─────────────────────────────────────┐
│ 第 1 步：注册 GitHub 账号          │
│ - 访问 github.com                   │
│ - 点击 Sign up 注册                 │
│ - 填写邮箱、密码、用户名            │
│ - 验证邮箱                          │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 2 步：创建代码仓库                │
│ - 点击右上角 + 号                   │
│ - 选择 New repository               │
│ - 仓库名：secret-reader             │
│ - 点击 Create repository            │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 3 步：下载 GitHub Desktop         │
│ - 访问 desktop.github.com           │
│ - 下载 Windows 版本                 │
│ - 双击安装，一直点 Next              │
│ - 启动并登录 GitHub                 │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 4 步：克隆仓库到本地              │
│ - 点击 File → Clone repository      │
│ - 选择 secret-reader 仓库           │
│ - 选择保存位置（桌面）               │
│ - 点击 Clone                         │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 5 步：复制代码文件                │
│ - 打开项目文件夹（20260311165212）  │
│ - 复制所有文件到仓库文件夹           │
│ - 确保这些文件都在根目录             │
│   • package.json                     │
│   • main.js                          │
│   • index.html                       │
│   • README.md                        │
│   • .gitignore                       │
│   • .github 文件夹                   │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 6 步：提交并上传代码              │
│ - 打开 GitHub Desktop                │
│ - 左下角输入：初次提交               │
│ - 点击蓝色按钮 Commit to main        │
│ - 点击右上角 Push origin             │
│ - 等待上传完成                       │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 7 步：等待自动打包（10-20分钟）   │
│ - 打开 GitHub 仓库                  │
│ - 点击 Actions 标签                 │
│ - 查看打包进度                       │
│ - 等待绿色对勾 ✅                    │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 8 步：下载打包文件                │
│ - 点击完成的任务（绿色对勾）         │
│ - 向下滚动找到 Artifacts             │
│ - 点击文件名下载 ⬇️                 │
│ - 保存到本地（下载文件夹）           │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 9 步：解压并安装                  │
│ - 找到下载的 .zip 文件              │
│ - 右键解压到桌面                     │
│ - 双击 .exe 文件                     │
│ - 按向导安装（一直点下一步）         │
│ - 完成后桌面出现图标                 │
└─────────────────────────────────────┘
  ↓
┌─────────────────────────────────────┐
│ 第 10 步：开始使用！                 │
│ - 双击图标启动软件                  │
│ - 调节透明度                         │
│ - 阅读小说                           │
│ - 老板来了？按 Alt + H 隐藏          │
└─────────────────────────────────────┘
  ↓
  完成！🎉
```

## 🎬 操作场景演示

### 场景 1：正常阅读

```
你正在办公...
  ↓
偷偷打开小说阅读器
  ↓
调节透明度到 70%（半透明）
  ↓
开启置顶，放在工作文档旁边
  ↓
边"工作"边看小说 😊
```

### 场景 2：老板突然出现（紧急隐藏）

```
正在看小说...
  ↓
老板来了！🙀
  ↓
按键盘：Alt + H
  ↓
窗口瞬间消失 ✅
  ↓
你继续假装工作 😎
```

### 场景 3：老板走了，继续看

```
老板离开...
  ↓
看向右下角托盘
  ↓
点击 📖 图标
  ↓
窗口重新出现 😊
  ↓
继续阅读...
```

## ⏱️ 时间预估

| 步骤 | 操作 | 预计时间 |
|------|------|----------|
| 1 | 注册 GitHub | 3 分钟 |
| 2 | 创建仓库 | 1 分钟 |
| 3 | 下载 GitHub Desktop | 2 分钟 |
| 4 | 克隆仓库 | 1 分钟 |
| 5 | 复制文件 | 1 分钟 |
| 6 | 上传代码 | 2 分钟 |
| 7 | 等待打包 | 10-20 分钟 ⏳ |
| 8 | 下载文件 | 2 分钟 |
| 9 | 解压安装 | 2 分钟 |
| **总计** | | **约 25-35 分钟** |

> 💡 提示：打包时间最长（10-20 分钟），其他步骤很快

## 🔄 修改代码后重新打包

```
修改代码（如换小说内容）
  ↓
GitHub Desktop 自动检测到变化
  ↓
输入提交信息，点击 Commit
  ↓
点击 Push 上传
  ↓
GitHub 自动开始新的一轮打包
  ↓
10-20 分钟后下载新版本
  ↓
重新安装覆盖旧版本
```

## 📝 检查清单

开始前，确保你已经准备好：

- [ ] 电脑已经联网
- [ ] 有一个可用的邮箱（QQ 邮箱也可以）
- [ ] 有足够的硬盘空间（至少 500MB）
- [ ] 稍微空闲的时间（30 分钟左右）

## 🎯 成功标志

当你完成所有步骤后，你应该拥有：

✅ GitHub 账号和仓库
✅ 桌面上有一个 **"隐蔽小说阅读器"** 图标
✅ 可以双击图标启动软件
✅ 可以调节透明度
✅ 可以通过 `Alt + H` 快速隐藏
✅ 可以通过托盘图标重新显示
