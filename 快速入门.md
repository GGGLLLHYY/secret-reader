# 🚀 3 分钟快速入门（最简版）

> 太长不看？看这里就够了！

---

## 步骤 1：注册 GitHub（3 分钟）

1. 打开 https://github.com 注册
2. 登录后点击右上角 **+** → **New repository**
3. 仓库名填写：`secret-reader`
4. 点击 **Create repository**

---

## 步骤 2：下载工具（2 分钟）

1. 访问 https://desktop.github.com/
2. 点击 **Download for Windows** 下载
3. 双击安装，一直点 **Next**

---

## 步骤 3：上传代码（3 分钟）

1. 启动 GitHub Desktop，登录你的 GitHub 账号
2. 点击 **File** → **Clone repository**
3. 选择 `secret-reader` 仓库，点击 **Clone**
4. 把 `C:\Users\QYT-099\CodeBuddy\20260311165212` 文件夹里的所有文件复制到克隆的文件夹
5. 在 GitHub Desktop 左下角输入：`初次提交`
6. 点击蓝色按钮 **Commit**
7. 点击右上角 **Push origin** 上传

---

## 步骤 4：等待打包（10-20 分钟）⏳

1. 打开你的 GitHub 仓库
2. 点击顶部的 **Actions** 标签
3. 等待任务完成（绿色对勾 ✅）
4. 点击任务，向下滚动找到 **Artifacts**
5. 点击下载图标 ⬇️

---

## 步骤 5：安装使用（2 分钟）

1. 解压下载的压缩包
2. 双击 `.exe` 文件安装
3. 双击桌面图标启动软件
4. 拖动滑块调节透明度
5. 点击 **"◻"** 或 **"🙈 快速隐藏"** 按钮，隐藏窗口

---

## 🎯 使用技巧

- **老板来了？** → 按 `Alt + H` 快速隐藏
- **继续看？** → 点击右下角托盘图标
- **退出软件？** → 右键托盘图标 → **退出**
